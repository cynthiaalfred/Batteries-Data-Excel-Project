# Batteries-Data-Excel-Project

### The purpose of this project is to demonstrate basic to intermediate Excel skills in data analysis, reporting, and dashboard creation. It shows how raw data can be transformed into meaningful business insights through visual reporting.
## Tools Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data visualisation techniques

### - Dashboard
#### - This Excel dashboard project was created to analyse battery sales data and present key business insights in a clear visual format. The project uses Pivot Tables, Pivot Charts, slicers, and interactive visuals to make the data easy to understand.

### - Insights Included
* Total sales performance
* Product category analysis
* Regional or customer-based trends
* Comparison of sales across different periods
* High-performing products and areas
